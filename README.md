# drivers-library
AVCommander Driver Library — ready-to-use JSON drivers for popular devices. Browse and download examples for Sony TVs, DSP Audio, and more. All drivers are fully customizable, or you can create your own from scratch directly in the AVCommander app. Build your perfect control system with drivers tailored to your needs.

To use a driver, download it to your iPhone or iPad, open the AVCommander app, go to Settings → Import From JSON Config File, select the file, and the device will appear in the DEVICES list.
